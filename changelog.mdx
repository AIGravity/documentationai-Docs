## Latest release

<Update label="Latest" description="Product foundation release" tags={["feature", "improvement", "bugfix"]}>

### New features

- Shipped the initial web, API, and AI service foundation so teams can build from a working product baseline. See the [API service foundation](/api-service-foundation) and [AI service foundation](/ai-service-foundation) docs.
- Added backend health checks for service status, making it easier to confirm whether core services are online. See [Health Checks](/health-checks).
- Introduced the Remix-based web app flow for routing, loading states, and the application shell. See [Frontend Application Shell](/frontend-application-shell).

### Updates

- Added Tailwind CSS v4 styling support for the web experience.

### Bug fixes

- Fixed the web container build so frontend deployments are more reliable. See [Docker Images and Packaging](/docker-images-and-packaging).

</Update>
