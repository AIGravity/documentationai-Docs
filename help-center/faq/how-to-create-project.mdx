## Overview

Creating a project lets you organize your documentation in Gravity.

## Steps

1. Sign in to your Gravity dashboard
2. Click "New Project" in the sidebar
3. Enter a name and (optionally) a description
4. Click "Create"
5. Your new project appears in the project list
