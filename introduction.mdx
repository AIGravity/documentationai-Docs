## Overview

Documentation.AI helps you author, organize, and publish documentation with MDX.

- Write content in MDX with reusable components
- Structure docs for tutorials, references, help, and changelogs
- Link related pages with cards and clear navigation
- Publish documentation that is easy to scan and maintain

<Callout kind="info">

This site focuses on documentation authoring and publishing workflows; it is not tied to any specific SaaS feature set.

</Callout>

## Documentation philosophy

Documentation works best when it matches the reader's task. Use concise pages for facts, step-based pages for procedures, and explanatory pages for concepts that need context.

Keep each page focused on one primary job, reuse shared pages instead of repeating the same guidance, and surface the next best action at the end of the page.
