````bash
npm install gravity@latest
````

````bash

````
</CodeGroup>

## Upgrading Gravity

Follow these steps to update your Gravity installation safely.

<Steps>
<Step title="Backup Data" icon="database">
Create a full backup of your project spaces and configurations.
</Step>

<Step title="Check Compatibility" icon="check-circle">
Review the breaking changes section in each release above.
</Step>

<Step title="Run Update" icon="download">
<CodeGroup tabs="Docker,CLI">
````bash
docker pull gravitydocs/gravity:latest
````

````bash
gravity upgrade
````
</CodeGroup>
</Step>

<Step title="Test Integration" icon="play">
Verify your API calls to `https://api.example.com/v1` endpoints.
</Step>
</Steps>

<Callout kind="tip">
After upgrading, clear your browser cache and restart services to ensure all changes take effect.
</Callout>
